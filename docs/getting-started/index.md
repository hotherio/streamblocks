# Getting Started

New to StreamBlocks? This section takes you from installation to your first custom block type in about fifteen minutes.

1. **[Installation](installation.md)** — install the library and optional provider extras.
2. **[Quickstart](quickstart.md)** — extract your first block from a text stream.
3. **[Your First Custom Block](first-custom-block.md)** — define a typed block with your own metadata and content models.

After that, read the [Concepts](../concepts/index.md) section to understand how the processing pipeline works, or jump straight to the task-oriented [Guides](../guides/index.md).
